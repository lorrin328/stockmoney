## 一、理论溯源

### 1.1 原作者

**研究员雷牛牛**，B站财经UP主，自称"名校硕士毕业、资深研究员"。其提出的"4%定投法"视频播放量超过28.6万次，后在微信公众号发布《4%定投法（强化版）》完善参数体系。

### 1.2 原版核心参数（强化版）

| 参数 | 值 | 说明 |
|------|-----|------|
| **总份数** | **25份** | 总资金平均分成25份 |
| **单标的上限** | **10份** | 同一只基金最多投入10份（即总资金的40%） |
| **触发基准** | **上一个买入点** | 非"2年最低点"，是严格的上次买入价 |
| **触发跌幅** | 4% | 从上一个买入点下跌>=4%触发买入1份 |
| **价格阶梯** | 0.96^n | 第n次买入价 = 第(n-1)次买入价 × 0.96 |
| **买入过滤** | E/P > 10% | 盈利收益率>10%（格雷厄姆低估线）才启动 |
| **卖出条件** | E/P < 6.4% | 盈利收益率<6.4%（格雷厄姆高估线）全部卖出 |
| **等待纪律** | 严格 | 未跌穿4%绝不提前买入 |
| **行业适配** | PE/PB区分 | 非周期看PE，周期股（银行/证券/半导体等）看PB |

### 1.3 核心操作口诀

```
资金分25，单只限10份
上一个买点，跌4买一份
E/P超10%，低估才启动
E/P低6.4%，高估全卖出
未跌穿4%，绝对不提前
只买跌不买涨，反弹不触发
```

## 二、适用场景与时机（核心！）

### 2.1 4%法适合的市场环境

| 市场环境 | 4%法表现 | 原因 | 推荐度 |
|----------|----------|------|--------|
| **震荡市** | ★★★★★ | 反复震荡，4%触发频繁，不断摊薄成本 | 最佳 |
| **下跌市/熊市** | ★★★★☆ | 越跌越买，左侧布局，成本摊薄效果最佳 | 很好 |
| **筑底期** | ★★★★★ | 反复探底，多次触发买入，积攒低价筹码 | 最佳 |
| **单边上涨市** | ★☆☆☆☆ | 不回调或回调<4%，大量现金闲置，严重踏空 | 不适用 |
| **V型反转** | ★★☆☆☆ | 深跌后快速反弹，可能只买1-2份就涨上去 | 较差 |

### 2.2 适用标的

| 标的类型 | 推荐度 | 理由 |
|----------|--------|------|
| **宽基指数ETF**（沪深300、中证500等） | ★★★★★ | 成立>7年，回撤可控（通常30-40%），PE/PB与点位高度相关 |
| 行业ETF（消费、医药等弱周期） | ★★★★☆ | 用PE估值，业绩相对稳定 |
| 行业ETF（银行、证券、半导体等强周期） | ★★★☆☆ | **必须用PB估值**，PE会失真 |
| 主动管理基金 | ★★★☆☆ | 需确认净值与估值指标的相关性 |
| **个股** | ☆☆☆☆☆ | 风险过高，不满足"成立>7年+业绩稳定"条件 |

### 2.3 不适合使用4%法的情况

1. **市场处于明确单边上涨通道**：如2024年9月后的A股反弹，4%法会完全空仓等待
2. **资金量过小**：25份设计意味着单份金额较小，小资金可能不够分散
3. **缺乏纪律性**：无法做到"跌了4%才买"、"未跌穿不买入"的人不适合
4. **追求高收益**：4%法是防守型策略，牛市中大概率跑输普通定投

## 三、原版 vs 修正版核心差异

### 3.1 我之前的理解错误

| 维度 | 我之前的错误理解 | 原版理论 |
|------|-----------------|----------|
| **份数** | 10份 | **25份**（单标的上限10份） |
| **触发基准** | "2年最低点" | **"上一个买入点"** |
| **买入条件** | 从低点跌4%就买 | **E/P>10% + 从上次买入点跌4%** |
| **卖出条件** | 我自创的分层止盈 | **E/P<6.4%**（格雷厄姆高估线） |
| **等待纪律** | 模糊 | **未跌穿4%绝不提前买入** |
| **首次买入** | 任意价格开始 | **设定观察价，等跌4%才买入** |

### 3.2 "上一个买入点" vs "2年最低点"的本质区别

这是理解4%法最关键的一点：

**错误理解（我之前）：**
- 价格从2年最低点跌4%就触发
- 假设低点是1.0，跌到0.96触发
- 反弹到1.1后，又跌回0.96——还是触发（因为0.96 < 1.0×0.96=0.96）

**原版理论（正确）：**
- 价格从"上一个买入点"跌4%才触发
- 第一次买入价1.0，触发线是0.96
- 如果价格跌到0.95买入了，新的触发线变为0.95×0.96=0.912
- 如果价格反弹到1.1，**不会触发**（因为1.1 > 0.912，但1.1不是从上一个买入点跌下来的）
- 只有当价格**从1.0这个买入点**跌4%到0.96，或**从后续买入点**再跌4%才触发

**关键区别**：原版只在"从上一次买入的价格往下跌"时才触发，而不是从"历史最低点"触发。这意味着反弹后需要更大的跌幅才能再次触发——纪律性更强，踏空风险也更大。

## 四、改进版设计

在原版修正版基础上，增加以下改进：

| 改进维度 | 原版 | 改进版 | 改进动机 |
|----------|------|--------|----------|
| **买入倍率** | 固定1份 | 0.5x-3x动态 | E/P越高买入越多 |
| **止盈策略** | E/P<6.4%全卖 | 分层止盈+移动止盈+E/P强制卖出 | 避免牛市卖飞 |
| **右侧补仓** | 无 | E/P>12%连续涨3天半份补仓 | 减少极端低估踏空 |
| **E/P代理** | 无 | 价格分位反向映射 | ETF无直接E/P数据 |

### 4.1 动态买入倍率

| E/P区间 | 估值状态 | 买入倍率 | 说明 |
|---------|----------|----------|------|
| E/P >= 12% | 极度低估 | 3.0x | 加大投入 |
| E/P 10%-12% | 低估 | 1.5x | 正常偏多 |
| E/P 8%-10% | 合理 | 1.0x | 常规定投 |
| E/P < 8% | 偏高 | 0.5x | 减少投入 |

### 4.2 分层止盈

| 盈利层级 | 触发条件 | 卖出比例 |
|----------|----------|----------|
| 第一层 | 盈利 >= 15% | 卖出持仓的30% |
| 第二层 | 盈利 >= 25% | 卖出持仓的30% |
| 第三层 | 盈利 >= 35% | 卖出持仓的40% |
| 移动止盈 | 从最高点回撤 >= 10% | 卖出剩余全部 |
| 强制卖出 | E/P < 6.4% | 卖出全部（无论盈利） |

## 五、回测验证结果

### 5.1 沪深300ETF（510300）回测

回测周期：2024-09-18 ~ 2026-04-30（约390天，单边上涨市）

| 策略 | 总收益率 | 年化收益 | 最大回撤 | 买入次数 | 卖出次数 | 资金利用率 |
|------|----------|----------|----------|----------|----------|----------|
| **原版4%定投法** | **+0.0%** | **+0.0%** | **0.0%** | **0** | **0** | **0%** |
| **改进版4%定投法** | **+0.0%** | **+0.0%** | **0.0%** | **0** | **0** | **0%** |
| 普通月定投 | +21.0% | +13.1% | 7.4% | 18 | 0 | 100% |

**结论**：在单边上涨市中，4%法严格空仓等待下跌，0交易。这是策略设计特性——**"只买跌不买涨"的纪律性代价就是牛市踏空**。

### 5.2 中证A500ETF（560610）回测

| 策略 | 总收益率 | 买入次数 | 卖出次数 | 资金利用率 |
|------|----------|----------|----------|----------|
| **原版4%定投法** | **+0.0%** | **0** | **0** | **0%** |
| **改进版4%定投法** | **+0.0%** | **0** | **0** | **0%** |
| 普通月定投 | +23.8% | 18 | 0 | 100% |

### 5.3 关键发现

1. **4%法在牛市中确实跑输普通定投**：这是策略设计的必然结果
2. **4%法在牛市中0回撤**：空仓意味着没有亏损风险
3. **资金利用率是关键差异**：4%法可能长期0%利用率，月定投100%
4. **适用场景决定策略选择**：
   - 如果你判断市场即将下跌或震荡 → 用4%法
   - 如果你判断市场将持续上涨 → 用普通定投或一次性买入

## 六、实盘操作手册

### 6.1 每日操作流程

```bash
# 1. 运行监控脚本
python scripts/investment_monitor.py

# 2. 查看生成的报告
data/signals/signal_report_YYYYMMDD.md

# 3. 重点关注以下信号：
#    - "4%定投-观察"：可开始观察，设定观察价
#    - "4%定投-买入"：触发买入条件
#    - "4%定投-卖出"：E/P<6.4%，全部卖出
#    - "4%定投-等待"：持仓中，等待下一次触发
```

### 6.2 首次使用步骤

1. **筛选标的**：选择成立>7年的宽基指数ETF
2. **判断估值**：确认E/P > 10%（可用监控系统的E/P代理值参考）
3. **设定观察价**：记录当前价格为观察基准
4. **等待触发**：等待价格跌到观察价 × 0.96
5. **触发买入**：买入1份（总资金的1/25）
6. **更新基准**：新的触发线变为买入价 × 0.96
7. **重复等待**：等待下一次触发

### 6.3 持仓管理

```
持仓状态：
  - 记录last_buy_price（上次买入价）
  - 计算触发线 = last_buy_price × 0.96
  - 每日检查：当前价 <= 触发线？
    - 是 → 检查E/P > 10%？
      - 是 → 买入1份
      - 否 → 继续等待
    - 否 → 继续等待
  - 同时检查E/P < 6.4%？
    - 是 → 全部卖出
    - 否 → 持有
```

### 6.4 心态管理

1. **接受踏空**：4%法的核心代价就是牛市不买入。"少赚比亏损强"
2. **不要追高**：即使市场涨了很多，只要未触发就不买入
3. **严格执行**：不要因"这次不一样"而提前买入
4. **长期视角**：4%法以2-3年为操作周期，不因短期波动改变
5. **分散配置**：25份分配到多只标的，避免单只标的黑天鹅

## 八、风险提示

1. **适用场景错误是最大的风险**：在单边上涨市使用4%法会严重踏空
2. **E/P代理不精确**：ETF无直接E/P数据，使用价格分位反向代理可能有偏差
3. **纪律性要求极高**：违反"未跌4%不买入"的规则会导致策略失效
4. **机会成本**：大量现金闲置期间可能错过其他投资机会
5. **历史不代表未来**：回测基于历史数据，市场环境变化可能导致策略失效
6. **参数敏感性**：4%触发比例、E/P阈值对结果影响大
