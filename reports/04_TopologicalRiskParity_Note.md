**作者：** Revant Nayar, El Mehdi Ainasse, Dnyanesh Kulkarni (FMI Technologies LLC)

**类型：** 投资组合构建 / 风险平价 / 网络方法

## 一、研究背景与动机

### 1.1 核心问题
传统投资组合构建方法在高维环境下存在根本缺陷：
- **均值-方差优化**（Markowitz）对协方差估计敏感，高维不稳定
- **层次风险平价**（HRP, Lopez de Prado 2016）虽通过聚类正则化解决了部分问题，但存在三个关键局限：
  1. **不支持多空组合**：HRP 仅设计为多头，无法处理带符号的截面信号
  2. **保守权重传播**：每个节点将权重完全传播给子节点（局部守恒），忽略了父节点与子节点之间的不完全相关性
  3. **缺乏经济先验**：层次结构完全由聚类 emergent 产生，无法施加市场/行业/因子等经济层级

### 1.2 市场直觉
被动投资和因子ETF的崛起，使市场回报结构呈现出天然的**树状层级**：
- 市场因子 → 行业因子 → 子行业/篮子 → 个股
- 这种嵌套共同成分主导了收益联动，使相关距离图具有强烈的层级结构
- 最小生成树（MST）恰好能提取这种稀疏骨干

## 二、核心方法论：Topological Risk Parity (TRP)

### 2.1 数学框架

**步骤1：活跃集筛选**
- 基于回望窗口的近期波动幅度阈值 ε 和信号阈值 τ 筛选活跃资产

**步骤2：构建相关距离图与MST**
- 使用 Mantegna 距离：$D_{ij} = \sqrt{(1-\rho_{ij})/2}$
- 在完全图上构建最小生成树，将 $n(n-1)/2$ 条边压缩为 $n-1$ 条边

**步骤3：选定根节点**
- Hub 模式：度最大的节点
- 最大信号模式：$|s_i|$ 最大的节点
- 固定索引模式：指定根节点（如SPY）

**步骤4：传播系数（核心创新）**

对任意内部节点 $u$，设其分支数为 $b(u)$，定义混合分裂-复制系数：

$$\alpha_u(\rho) := (1-\rho) + \frac{\rho}{b(u)}, \quad \rho \in [0,1]$$

**步骤5：拓扑因子递归**
- 根节点：$g_r = 1$
- 非根节点：$g_v = \alpha_{pa(v)}(\rho) \cdot g_{pa(v)}$
- 路径公式：$g_v = \prod_{u \in Path(r,v)} \left[(1-\rho) + \frac{\rho}{b(u)}\right]$

**步骤6：最终权重**
- 预归一化组合：$x_v = s_v^{(A)} \cdot g_v$
- 杠杆归一化：$w_v = L \cdot \frac{x_v}{\|x\|_1}$

### 2.2 两个变体

| 特征 | TRP-MST（变体I） | TRP-SPY/XL（变体II，半监督） |
|------|------------------|------------------------------|
| 根节点 | 启发式或固定 | 虚拟市场根节点（SPY） |
| 第二层 | 自然涌现 | 11个行业ETF（XL*） |
| 经济先验 | 有限 | 强 |
| 后处理 | 无 | 截断(clip) + 阈值(prune) |

**半监督变体的关键构造：**
- 引入虚拟市场根节点 $m$（信号为0，不分配权重）
- 将 $m$ 连接到所有行业ETF
- 对非行业节点运行DFS提取生成树
- 当 $|X| > 1$ 时，增强图含环，DFS遍历顺序会影响非行业节点的父节点分配

## 三、关键理论结果

### 3.1 极限情况（命题3）
- **ρ = 0**：退化为归一化原始信号分配（无拓扑衰减）
- **ρ = 1**：退化为保守等分树方法（HRP-like）
- **ρ ∈ (0,1)**：连续插值，拓扑以连续方式进入分配

### 3.2 局部质量放大（命题4）—— 最深刻的理论结果

对分支数为 $b(u)$ 的内部节点：

$$\sum_{v \in Ch(u)} g_v = \beta_u(\rho) \cdot g_u, \quad \beta_u(\rho) := b(u)(1-\rho) + \rho$$

**关键推论：**
- $\beta_u(\rho) \geq 1$，等号成立当且仅当 $b(u)=1$ 或 $\rho=1$
- **当 $\rho < 1$ 且 $b(u) > 1$ 时，分支会扩展总后代因子质量**
- 这意味着：沿单条路径因子递减（$\alpha_u \leq 1$），但跨后代聚合时质量可以放大
- **这是TRP与HRP的根本区别**：HRP在每个节点保守分裂，TRP允许非保守的局部质量放大

### 3.3 层级质量上界（推论2）
设 $B$ 为最大分支数，$\Gamma_B(\rho) = B(1-\rho) + \rho$，则深度 $\ell$ 的节点总因子质量：

$$\sum_{v \in L_\ell} g_v \leq \Gamma_B(\rho)^\ell$$

### 3.4 信号保持（命题5）与标度对称（命题6）
- $sign(w_v) = sign(s_v^{(A)})$：原始信号的符号被保留
- 正标度不变：$w(cs) = w(s)$；负标度翻转：$w(cs) = -w(s)$

### 3.5 条件Lipschitz稳定性（定理1）
固定根树和拓扑因子时，映射 $F(s) = L \cdot D_g s / \|D_g s\|_1$ 满足：

$$\|F(s) - F(s')\|_1 \leq \frac{2L}{\gamma} \|s - s'\|_1$$

**重要提醒**：此稳定性是**条件性**的——当MST结构、根节点或活跃集发生组合变化时，不连续性真实存在。

## 四、与HRP的概念对比

| 维度 | TRP-MST | TRP-SPY/XL | 经典HRP |
|------|---------|------------|---------|
| 主结构 | 有根MST | 有根生成树（市场/行业先验） | 二叉树状图 |
| 多空支持 | 是 | 是 | 否 |
| 根节点 | 启发式/固定 | 虚拟市场根 | 无外生根 |

| 需协方差逆 | 否 | 否 | 否 |
| 需聚类方差递归 | 否 | 否 | 是 |
| 可施加经济先验 | 有限 | 强 | 有限 |

## 五、实践意义与优势

### 5.1 相对HRP的八大优势
1. **显式层级控制**：半监督变体可直接编码市场/行业/股票本体论
2. **多空组合支持**：天然适合市场中性、统计套利、L/S趋势策略
3. **经济可解释性**：反映Barra/Axioma式的因子层级结构
4. **稀疏且可解释**：依赖树而非满协方差矩阵
5. **平滑插值参数**：ρ提供连续调节杠杆
6. **无需协方差逆和聚类方差递归**：管道更简单
7. **后处理兼容**：截断、阈值、杠杆目标、行业中性化易于附加
8. **宏观冲击韧性**：经济动机树确保危机时同行业/因子内相关性飙升不会导致过度集中

### 5.2 局限性与注意事项
1. **根节点依赖**：不同根节点选择可能彻底改变分配
2. **MST不稳定性**：相关性的微小变化可能组合性地改变MST
3. **信息损失**：MST仅保留n-1条边，丢弃了满图信息
4. **半监督构造的顺序依赖**：当多个行业ETF连接虚拟根时，非行业节点的父节点分配依赖DFS遍历顺序
5. **回退分支的工程性**：零收益虚拟根作为增强相关矩阵的节点是务实但非理论理想的构造

## 六、三层嵌套流模型（第3节）

论文提供了一个简洁的**风格化机制**，解释为何被动/因子流创造树状结构：

**收益分解：**
$$r_i = \theta_M z_M + \theta_S z_{s(i)} + \theta_B z_{b(i)} + \varepsilon_i$$

**隐含协方差层级：**
- 同篮子：$Cov = \theta_M^2 + \theta_S^2 + \theta_B^2$
- 同行业不同篮子：$Cov = \theta_M^2 + \theta_S^2$
- 不同行业：$Cov = \theta_M^2$

**相关距离严格排序：** $D_{\text{same basket}} < D_{\text{same sector}} < D_{\text{different sector}}$

**被动/因子主导旋钮 λ：**
- λ → 0：相关性崩塌，MST更受样本噪声驱动
- λ → 1：层级清晰分离，MST骨干更稳定

**星型 vs 深层层级：**
- 若 $\theta_M^2 \gg \theta_S^2, \theta_B^2$：MST退化为星型（单一市场枢纽）
- 深层层级需要多层共同流共存

## 七、批判性思考

### 7.1 理论贡献评价
- **新颖性**：将信号传播与拓扑结构结合的思路具有原创性，特别是非保守传播法则
- **严密性**：理论推导清晰，从基本定义到稳定性定理层次分明
- **局限性**：条件稳定性定理回避了最困难的问题——MST变化时的组合不连续性

### 7.2 实践可行性
- **优势场景**：多空统计套利、市场中性策略、因子中性化要求高的策略
- **挑战**：根节点选择缺乏客观标准；MST对估计误差敏感；半监督变体的顺序依赖需要额外处理
- **与HRP的关系**：TRP并非HRP的替代，而是互补——当需要多空支持和经济先验时选用TRP

### 7.3 与本文集其他论文的关联
- 与**论文03（Spurious Predictability）**关联：TRP的稀疏性可能减少伪预测性带来的过拟合
- 与**论文05（Investing Is Compression）**关联：TRP可视为一种"压缩"协方差矩阵信息到树结构的方法
- 与**论文06（Strategy Backtests）**关联：需要在严格回测框架下验证TRP的实际表现

## 九、投资启示

1. **组合构建不应只关注信号强度，还应考虑信号在相关结构中的位置**——深度嵌入拥挤集群的信号应被衰减
2. **ρ参数是强大的风险调节杠杆**——从纯信号驱动(ρ=0)到完全拓扑保守(ρ=1)的连续谱
3. **经济层级先验比纯统计聚类更具危机韧性**——2008年和2020年的经验表明，样本内低相关在冲击下会急剧飙升
4. **多空策略的权重分配必须保留信号符号**——任何将信号转化为方差或风险预算的方法都可能扭曲投资意图
