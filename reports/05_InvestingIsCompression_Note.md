## 一、历史脉络与核心命题

### 1.1 一段被埋没的思想史

本文梳理了从1956年至今的一条**被主流经济学排斥的思想谱系**：

```
Kelly (1956, Bell Labs)
    ↓ 信息论视角提出对数财富最大化
Thorp (1960s-70s)
    ↓ 应用于21点→Princeton-Newport Partners量化基金
Shannon → Samuelson (MIT, 1960s晚间会议)
    ↓ Samuelson "对数效用有本能厌恶"，成功将其排斥于主流经济学之外
Cover (Stanford, 1980s-90s)
    ↓ 证明Kelly准则是客观最优，非主观效用函数
本文作者
    ↓ 揭示Kelly目标函数的信息论结构：投资 = 压缩问题
```

**Samuelson的反对**：认为对数效用是"任意的、主观的"，过于保守。他成功将这一思想标记为"从香农信息论借来的谬误"。

**Cover的反驳**（分三层）：
1. 最大化长期财富
2. 最小化破产风险
3. **博弈论意义上的竞争性最优**——即使短期亦然（最关键的反驳）

### 1.2 核心命题

> **投资本质上是压缩问题。**

Kelly的目标函数（最大化期望对数回报）将投资问题分解为三项：
- **金钱项**：游戏规则/回报结构
- **熵项**：不确定性（与策略无关）
- **散度项**：策略分布与真实分布的KL散度

**关键洞察**：前两项在给定回测中对所有策略是常数。**唯一最大化复合增长的方式是最小化散度项**——即让策略分布尽可能接近真实（未知）分布。

## 二、从经典Kelly到一般形式的推导

### 2.1 经典Kelly（二元赌局）

**设定**：概率 $p$ 赢得回报乘数 $r$，概率 $1-p$ 输掉全部赌注。

**最优下注比例**（edge/odds公式）：

$$f^* = \frac{pr - 1}{r - 1}$$

**最大增长率**：

$$g^* = p\log(pr) + (1-p)\log\frac{(1-p)r}{r-1}$$

**信息论解释**：当市场概率 $q$ 满足 $qr=1$（公平赔率）时：

$$g^* = p\log\frac{p}{q} + (1-p)\log\frac{1-p}{1-q} = D_{KL}(P \| Q)$$

**惊人结论**：最优增长率等于我们的分布与市场分布之间的KL散度。

### 2.2 赛马市场（Horse Race Market）—— 关键过渡

**设定**：$n$ 匹马，每轮只有一匹获胜。$w_i$ 为第 $i$ 匹马的赌注比例，$p_i$ 为获胜概率，$r_i$ 为回报乘数。

**期望对数回报**：

$$g(W) = \sum_{i=1}^N p_i \log(r_i w_i)$$

**核心分解**（本文最重要的数学推导）：

$$g(W) = \underbrace{\sum_{i=1}^N p_i \log(r_i)}_{\text{金钱项}} - \underbrace{H(P)}_{\text{熵项}} - \underbrace{D_{KL}(P \| W)}_{\text{散度项}}$$

**三句话理解这个分解**：
1. **金钱项**：游戏规则决定，与我们的分配 $W$ 无关
2. **熵项**：真实分布的不确定性，与 $W$ 无关
3. **散度项**：唯一可操控的项，且 $D_{KL} \geq 0$，等号当且仅当 $W = P$

**反直觉结论**：最优策略是按**真实概率**分配赌注，与回报 $r_i$ 无关！（在公平赔率下 $q_i r_i = 1$）

**更一般的表达**：

$$g(W) = D_{KL}(P \| Q) - D_{KL}(P \| W)$$

- $D_{KL}(P \| Q)$：市场错误定价的程度（赚钱空间）
- $D_{KL}(P \| W)$：我们离最优分配的距离（摩擦损失）

**核心直觉**：你不能在定价准确的市场赚钱，也不能简单地同意市场。你必须**不同意市场，同时比市场更接近真实**。

### 2.3 一般多资产情况 —— Cover的技巧

**困难**：多资产组合中每期的回报是 $\sum_i w_i r_{i,t}$（和的内积），对数内部有多个非零项，无法直接分解。

**Cover的惊人技巧**：**积的和 = 和的积的展开**

以两资产为例：

$$(w_a r_{a,1} + w_b r_{b,1})(w_a r_{a,2} + w_b r_{b,2})$$
$$= w_a^2 r_{a,1}r_{a,2} + w_b^2 r_{b,1}r_{b,2} + w_a w_b r_{a,1}r_{b,2} + w_b w_a r_{b,1}r_{a,2}$$

**解释框架转换**：
- 传统：每期限加权回报相乘
- Cover：将资金预分配到**指数级数量的多期单资产序列**上，每个序列内不换手

**序列权重**：对于频率为 $P = (p_1, ..., p_m)$ 的序列，权重为 $\prod_i w_i^{np_i}$

**类型类（Type Class）集中**：
- 使用Stirling近似：$|T_n(W)| = e^{nH(W)}$
- 每个序列权重：$w_s = e^{-nH(W)}$
- 类型类总权重 ≈ $e^{-nH(W)} \cdot e^{nH(W)} = 1$

**精确KL边界**：非主导类型类的权重以指数衰减：

$$\text{总权重} = e^{-n D_{KL}(P \| W)}$$

**最终分解**（一般情况）：

$$g(W) = \log \bar{R}_{T(W^*)} - H(W^*) - D_{KL}(W^* \| W)$$

再次确认：**投资 = 金钱项 - 熵项 - 散度项**

## 三、实践应用

### 3.1 回测与策略比较

**问题**：市场数据非平稳，极端事件使对数目标的平均值难以统计解释。

**解决方案**：由于金钱项和熵项在回测中恒定，两策略的增长率差为：

$$g(W_B) - g(W_A) = D_{KL}(W^* \| W_A) - D_{KL}(W^* \| W_B)$$

**意义**：策略差异可用**比特（bits）**精确度量——这类似于统计力学中的吉布斯自由能。

### 3.2 MDL、市场预测与策略表现

**信息论 vs 统计学习理论**：
- 统计学习理论：依赖从平稳源重采样来检测过拟合
- **信息论/MDL原则**：随机性不可压缩，泛化就是寻找简洁的压缩描述

**核心洞见**：当策略表现和预测表现都被视为信息论问题时，信号改进与策略改进之间的关系更加直接。

### 3.3 Winner Fraction启发式 —— 原创贡献

**问题场景**：VC投资，需要在数年前决定资本分配，无法精确建模联合收益分布。

**启发式**：按每资产在候选集中**占优的概率**（winner fraction）分配资本：

$$W'_i = P(\text{资产 } i \text{ 在候选集中表现最优})$$

**理论保证**：

$$g(W^*) - g(W') \leq H(W')$$

其中 $H(W')$ 是winner fraction分布的熵。

**解读**：
- 当熵低（分布极端，少数资产极大概率胜出）时，启发式接近最优
- 当候选集小或回报分布极端时尤其有用
- 无需建模实际回报，只需估计"谁赢"的概率

## 四、深层哲学含义

### 4.1 风险-收益框架的反思

传统金融（Markowitz MPT）：风险与收益是两个正交轴——在给定风险下最大化收益。

**本文批判**：
1. 即使市场是高斯分布，MPT仍次优——因为Kelly目标是唯一最优
2. 长尾问题（Taleb）是统计关切，但更深层次的问题是**框架错误**

### 4.2 投资的信息论本质

| 维度 | 传统视角 | 信息论视角 |
|------|---------|-----------|
| 核心问题 | 风险-收益权衡 | 分布匹配/压缩 |
| 最优准则 | 效用最大化（主观） | 对数最优（客观） |
| 策略比较 | 夏普比率、t统计 | KL散度（比特） |
| 不确定性 | 方差 | 熵 |
| 错误代价 | 机会成本 | 散度摩擦 |

### 4.3 与本文集其他论文的关联

- **论文03（Spurious Predictability）**：过拟合本质上是在拟合噪声（不可压缩的随机性），信息论框架天然对抗过拟合
- **论文04（TRP）**：TRP将协方差信息"压缩"到树结构，可视为对投资问题的一种压缩表示
- **论文06（Strategy Backtests）**：回测评估应从统计显著性转向信息论度量（比特节省）

## 五、批判性评价

### 5.1 优势
1. **理论优美**：将投资问题与信息论的核心概念（熵、散度、压缩）建立深刻联系
2. **实践指导**：提供了具体的回测比较方法和winner fraction启发式
3. **历史澄清**：纠正了Samuelson对Kelly准则的误解

### 5.2 局限与挑战
1. **真实分布未知**：$W^*$ 永远无法知道，所有结果都是相对某个估计的
2. **序列独立性假设**：Cover的技巧隐含假设回报序列可类型化，对非独立同分布过程是否适用存疑
3. **交易成本忽略**：universal portfolio被普遍认为因交易成本高而不实用
4. **多期vs单期**：实际投资中再平衡是连续的，"序列"概念是理论构造

### 5.3 待深入问题
- 如何在非i.i.d.、非平稳环境中应用信息论框架？
- 压缩视角对期权定价、利率衍生品等复杂工具有何启示？
- 深度学习中的信息瓶颈（Information Bottleneck）与投资压缩有何联系？

## 六、核心公式速查

| 公式 | 名称 | 场景 |
|------|------|------|
| $f^* = (pr-1)/(r-1)$ | Kelly公式 | 二元赌局 |
| $g^* = D_{KL}(P \| Q)$ | 增长率=KL散度 | 公平赔率 |
| $g(W) = \sum p_i \log(r_i) - H(P) - D_{KL}(P \| W)$ | 三项分解 | 赛马市场 |
| $g(W) = D_{KL}(P \| Q) - D_{KL}(P \| W)$ | 双散度形式 | 一般市场 |
| $g(W^*) - g(W') \leq H(W')$ | 熵上界 | Winner Fraction启发式 |

## 七、对投资实践的直接启示

1. **任何策略评估都应问**：该策略的分布与真实分布的KL散度是多少？能否用比特表达改进？
2. **当无法估计完整联合分布时**，估计"谁赢"的概率比估计具体回报更可行
3. **分散化的信息论解释**：分散化不仅是降低方差，更是**降低对错误分布的暴露**
4. **市场有效性的信息论版本**：市场越有效，$D_{KL}(P \| Q)$ 越小，主动管理的理论空间越小
