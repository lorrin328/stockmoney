## 一、核心贡献

1. **伪造审计**：将完整预测工作流与合成参考类测试
2. **绝对幅度差距（ΔZ）**：量化选择导致的绩效膨胀
3. **有效多重性（K_eff）**：在相关搜索下校正极端值膨胀

## 二、虚假性级联

```
数据泄漏 → 选择偏误 → 规格错误
```

## 三、核心方法论

- **全局零假设**：$E[r_t | \mathcal{F}_{t-1}] = 0$
- **有效多重性**：$K_{eff} = K^2 / \sum \lambda_i^2$
- **缩放律**：$E[Z_{IS}^*] = \Theta(\sqrt{\log K_{eff}})$
- **样本外有界性**：$Z_{WF}^* = O_p(1)$
- **诊断指标**：ΔZ = $Z_{IS}^* - Z_{WF}^*$（τ不变，跨研究可比）

## 四、投资启示

1. 任何工作流必须先通过伪造审计
2. ΔZ比原始夏普更诚实
3. 不相交walk-forward是黄金标准
