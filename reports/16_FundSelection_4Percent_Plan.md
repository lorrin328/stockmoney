## 四、4%定投执行方案

**总资金：** 90万元  
**定投周期：** 18个月（每月5万）  
**预留机动：** 0元（全额按计划投入，极端情况再调整）

### 4.2 动态加码规则

当某一标的触发以下条件时，将该标的当月额度 **加倍**（从其他标的匀出或动用预留资金）：

| 触发条件 | 加码标的 | 加码幅度 |
|----------|----------|----------|
| 沪深300 PE-TTM < 12x | 沪深300ETF + 中证A500 | +100% |
| 恒生科技 PE < 16x | 恒生科技ETF | +100% |
| 光伏ETF净值创2年新低 | 光伏ETF | +50% |
| 创新药ETF净值 < 0.8元 | 创新药ETF + 港股创新药 | +50% |
| 红利低波股息率 > 6% | 红利低波ETF | +50% |
| 银行ETF PB < 0.45x | 银行ETF | +50% |

### 4.3 估值监控与再平衡

**每月定投日检查清单：**

1. **估值检查**
   - 沪深300 PE-TTM 当前值 vs 12x/18x阈值
   - 恒生科技 PE 当前值 vs 16x/25x阈值
   - 各持仓标的净值 vs 2年高低点分位

2. **偏离度检查**
   - 任一标的大类（宽基/核心/卫星）偏离目标±5% → 下个月调整定投比例
   - 任一单个标的偏离目标±10% → 触发再平衡

3. **事件检查**
   - 十五五规划具体政策落地 → 评估对相关标的的影响
   - 行业重大政策变化（如集采、补贴退坡）→ 评估持仓逻辑是否破坏

**每季度强制操作：**
- 卖出涨幅最大且估值偏高的标的盈利部分
- 加仓跌幅最大且基本面未变的标的

**每年12月强制再平衡：**
- 将所有标的恢复至目标仓位
- 评估下一年度十五五政策方向变化

### 4.4 退出与止盈规则

| 条件 | 操作 |
|------|------|
| 单只标的浮盈 > 50% | 卖出50%盈利，本金继续定投 |
| 沪深300 PE-TTM > 20x | 宽基仓位减半，转入红利低波 |
| 恒生科技 PE > 35x | 恒生科技仓位减半 |
| 十五五政策方向重大调整 | 重新评估相关标的持仓逻辑 |
| 达到4%年化目标且需用钱 | 逐步赎回，优先赎回高估标的 |

## 八、总结：为什么这个组合能实现4%+

1. **估值安全边际**：底仓（沪深300/恒生科技）均处于历史20-25%分位，下行空间有限
2. **产业趋势明确**：新能源、创新药、机器人均处于十五五规划核心方向，政策+产业双轮驱动
3. **高股息托底**：红利低波+银行提供5%+股息率，即使市场横盘也能累积收益
4. **全球化分散**：恒生科技覆盖平台经济+AI应用，与A股相关性约0.6，降低单一市场风险
5. **定投纪律**：18个月分批买入，避免择时焦虑，自动实现"高点少买、低点多买"
6. **动态加码**：在极端低估时加倍投入，提升长期回报

**历史参照**：2018年底沪深300 PE 10.5x时开始定投，3年后年化收益约12%。当前估值虽不如2018年极端，但已具备足够的安全边际。
